# Phase 9 Recommendation

Recommended Phase 9:

Add optional alternate CPU telemetry provider support and first user-controlled sensor calibration tools. Keep all tuning writes blocked. Focus on HWiNFO shared-memory detection/readout if available, Windows counter fallback for CPU frequency/load, manual sensor pinning improvements, sensor naming overrides, and exportable telemetry diagnostics. Do not enable MSI, NVIDIA, fan, EC, service, startup, scheduled task, registry, or automatic game-apply writes.
