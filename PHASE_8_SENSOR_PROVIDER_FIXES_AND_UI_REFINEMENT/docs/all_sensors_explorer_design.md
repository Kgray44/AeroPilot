# All Sensors Explorer Design

The All Sensors Explorer remains the complete raw-data view.

It includes:

- Full-width search.
- Filter row for hardware type, sensor type, category, selected/headline, problems only, and favorites only.
- Row count such as `Showing 149 of 149 sensors`.
- Pin selected sensor.
- Unpin selected sensor.
- Clear filters.
- Export raw sensor snapshot.

The table keeps important columns first:

- Favorite.
- Selected.
- Category.
- Hardware.
- Sensor type.
- Name.
- Value.
- Unit.
- Notes.

Source, hardware type, min, max, normalized key, and score remain available later in the row.
